HTML形式の1カラムのメールを作成してみました。

投稿日2019.02.26
  1. ホーム
  2. 備忘録
  3. 2019/02/26記事

HTML冒頭は文章型宣言、html要素、head要素を記述します。

2/25（月）の記事でまとめのおさらいになりますが、文章型宣言、HTML開始宣言を以下のように記述していきます。

▼文章型宣言、HTML開始宣言の記述

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd”> 
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="ja”>

続くhead要素では、文字コード、ページタイトル、viewportを設定します。

▼文字コード、ページタイトル、viewportの記述

  <meta http-equiv="Content-Type" content="text/html;charset=UTF-8" /> 
  <meta name="viewport" content="width=device-width,initial-scale=1.0" /> 
  <title></title>

body要素にはメール本文を設定します。

Body要素に入るメール本文は2/25（月）の記事でもまとめたとおり、テーブルレイアウトで記述していきます。 その際、table要素の幅、セル内の余白設定はtebleタグ内に記述することも忘れずに行います。

▼body要素の記述

<body>
  <table width="100%" cellpadding="20" cellspacing="0">
    <tr>
      <td>
        <h1>Responsive Email Letter</h1>
        <p>レスポンシブEメールに関する情報を提供します。</p>
        <h2>New Devices</h2>
        <p>新しいスマートフォンが数機種発売されています。CSSのサポート状況については、情報が入り次第お届けします。</p>
      </td>
    </tr>
  </table>
</body>

上記記述をひとつのファイルにまとめたものが下記になります。

▼1カラムHTMLメールのサンプル

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="ja">
<head>
  <meta http-equiv="Content-Type" content="text/html;charset=UTF-8" />
  <meta name="viewport" content="width=device-width,initial-scale=1.0" />
  <title></title>
</head>
<body>
  <table width="100%" cellpadding="20" cellspacing="0">
    <tr>
      <td>
        <h1>Responsive Email Letter</h1>
        <p>レスポンシブEメールに関する情報を提供します。</p>
        <h2>New Devices</h2>
        <p>新しいスマートフォンが数機種発売されています。CSSのサポート状況については、情報が入り次第お届けします。</p>
      </td>
    </tr>
  </table>
</body>
</html>

CSSはhead要素内に記述します。

HTMLメールでは、外部ファイルのCSSを読み込めないケースを避けるため、html内にCSSの記述を行います。
方法としてはhead要素内に、<style type=“text/css”>と</style>に囲まれたstyle要素を追加して、この中に記述を行います。前述のサンプルメールにCSSの記述を行なったものが下記になります。

▼1カラムHTMLメールのサンプル（CSS記述入り）

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="ja">
<head>
  <meta http-equiv="Content-Type" content="text/html;charset=UTF-8" />
  <meta name="viewport" content="width=device-width,initial-scale=1.0" />
  <title></title>
  <style type="text/css" >
  body {
    font-family: sans-serif;
    font-size: 17px;
    padding: 0;
    margin: 0;
    -webkit-text-size-adjust: 100%;
    -ms-text-size-adjust: 100%;
  }
  table, td {
    border-collapse: collapse;
    mso-table-lspace: 0;
    mso-table-rspace: 0;
  }
  a {
    text-decoration: underline;
  }
  img {
    border: 0 none;
    height: auto;
    line-height: 1;
    outline: none;
    text-decoration: none;
  }
  a img {
    border: 0 none;
    text-decoration: none;
  }
    h1, h2, h3, h4, h5, h6 {
    margin: 0;
    line-height: 1.1;
  }
  h1 {
    font-size: 24px;
  }
  h2 {
    font-size: 22px;
  }
  h3 {
    font-size: 20px;
  }
  h4 {
    font-size: 18px;
  }
  h5 {
    font-size: 16px;
  }
  p {
    color: #333;
    font-size: 17px;
    line-height: 1.47;
  }

  </style>
</head>
<body>
  <table width="100%" cellpadding="20" cellspacing="0">
    <tr>
      <td>
        <h1>Responsive Email Letter</h1>
        <p>レスポンシブEメールに関する情報を提供します。</p>
        <h2>New Devices</h2>
        <p>新しいスマートフォンが数機種発売されています。CSSのサポート状況については、情報が入り次第お届けします。</p>
      </td>
    </tr>
  </table>
</body>
</html>

CSSをインラインスタイルに変換します。

style要素に記述したCSSはメーラーによっては反映されない(Gmail)ことがあります。 そのため、CSSの記述はインラインスタイル（各要素の開始タグ内に直接CSSのコードを書き込んでいくこと）に変換します。

インライスタイルへの変換は手動だと面倒になりますので、ここでは「CSS Inliner Tool」という自動変換サービスを使用します。 使用するhtmlソースをコピペして、変換ボタンをクリックすると自動で変換後のソースが表示されます。

▼インラインスタイル変換後のソース(style要素内のCSS記述は削除済)

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" xml:lang="ja">
<head>
  <meta http-equiv="Content-Type" content="text/html;charset=UTF-8">
  <meta name="viewport" content="width=device-width,initial-scale=1.0">
  <title></title>
</head>
<body style="font-family: sans-serif;font-size: 17px;padding: 0;margin: 0;-webkit-text-size-adjust: 100%;-ms-text-size-adjust: 100%;">
  <table width="100%" cellpadding="20" cellspacing="0" style="border-collapse: collapse;mso-table-lspace: 0;mso-table-rspace: 0;">
    <tr>
      <td style="border-collapse: collapse;mso-table-lspace: 0;mso-table-rspace: 0;">
        <h1 style="margin: 0;line-height: 1.1;font-size: 24px;">Responsive Email Letter</h1>
        <p style="color: #333;font-size: 17px;line-height: 1.47;">レスポンシブEメールに関する情報を提供します。</p>
        <h2 style="margin: 0;line-height: 1.1;font-size: 22px;">New Devices</h2>
        <p style="color: #333;font-size: 17px;line-height: 1.47;">新しいスマートフォンが数機種発売されています。CSSのサポート状況については、情報が入り次第お届けします。</p>
      </td>
    </tr>
  </table>
</body>
</html>

画像を挿入する。
