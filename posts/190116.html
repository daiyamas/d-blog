お問い合わせフォームは紐付けられているcontact_me.php、contact_me.jsファイルの方で制御されています。 phpファイルには送信先のメールアドレスを設定します。 Jsファイルには何らかの原因で送信できなかった場合のアラート表示の文言が設定されていましたので こちらも日本語に変更していきます。

▼ 変更前のcontact_me.js

// Create the email and send the message
$to = 'daiyamas@gmail.com'; // Add your email address inbetween the '' replacing yourname@yourdomain.com - This is where the form will send a message to.
$email_subject = "Website Contact Form:  $name";
$email_body = "You have received a new message from your website contact form.\n\n"."Here are the details:\n\nName: $name\n\nEmail: $email_address\n\nPhone: $phone\n\nMessage:\n$message";
$headers = "From: noreply@yourdomain.com\n"; // This is the email address the generated message will be from. We recommend using something like noreply@yourdomain.com.
$headers .= "Reply-To: $email_address";   
mail($to,$email_subject,$email_body,$headers);
return true;

シンタックスハイライトに不具合が発生。

html、css、jsファイルのソースをページ内で表示させる際、Google code-prettifyを使用しているのですが、 css、js形式のソースを表示させようとしたところ「htmlファイル内の構文に間違いがある」とアラートが出てしまいます。 このページではひとまず、ハイライト表示のない仕様で貼り付けておきましたが問題が発生する原因の検証が必要です。 スマホで表示させた際にはテキストのエンコードがゴシック系ではなくRoman系に変換されるなどの不具合も確認しているため、 別の方法での仕様変更も視野に入れて改善をして行きます。
